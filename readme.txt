== Installation ==

1. Download the anti-hacking-tools.zip file to your computer.
2. Login Admin, Installed new plugins, click "add new" then upload and "install now".
3. Or unzip it, use ftp upload the "anti-hacking-tools" folder (including all files within) to your "/wp-content/plugins/anti-hacking-tools/"
4. Activate the plugin through the 'Plugins' menu in WordPress
5. in Tab Plugin click "Anti Hacking Tools" 
6. Checklist all for best security and click "Save Changes"
7. Enjoy!
